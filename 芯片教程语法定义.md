## 1. 静态声明区 (Static Scope)

在 `if __name__ == "__main__":` 之前只允许：

1. 定义 `INPUT`。
2. 声明全局变量并赋初值。
3. 定义显式常量。

禁止在静态区写 `OUTPUT`、计算表达式、连线逻辑。

```python
hp: Number = 100
target: Entity = None
names: List[String] = ["A", "B"]
PI: Final[Number] = 3.14
speed_in = INPUT("Speed", "Number")
```

## 2. 主逻辑区 (Main Block)

所有计算和连线必须写在 `if __name__ == "__main__":` 内。

```python
if __name__ == "__main__":
    new_hp = SUBTRACT(hp, 10)
    SET(hp, new_hp, is_pressed)
```

## 3. 位置参数规则 (Only Positional)

### 3.1 节点创建

统一格式：

`变量名 = 模块名(输入1, 输入2, 输入3, ...)`

示例：

```python
sum_val = ADD(a, b)
vec = Combine(x, y, z, w)
out = OUTPUT(sum_val, "Result")
```

### 3.2 INPUT/OUTPUT 固定格式

- `INPUT(name, data_type)`
- `OUTPUT(value, name, [data_type])`

示例：

```python
a = INPUT("MySpeed", "Number")
b = INPUT("Target", 1)
OUTPUT(a, "SpeedOut")
```

### 3.3 变量读写

- 读取：直接使用变量名。
- 写入：`SET(变量名, 新值, [触发信号])`。
- 触发信号省略时默认 `1.0`。

### 3.4 端口引用

1. `var["端口名"]`
2. `var[n]`（从 0 开始）
3. 裸变量（仅当节点只有一个输出）

### 3.5 字面量

在函数参数里直接写字面量，会自动转成 `Constant` 节点：

```python
scaled = MULTIPLY(hp, 2.0)
text_ok = EQUAL(name_a, "target")
vec = Combine(1.0, 2.0, 0.0, 0.0)
```

## 4. DataType 规范

支持类型名：

- 标量：`"Entity"`, `"Number"`, `"String"`, `"Vector"`
- 数组：`"ArrayNumber"`, `"ArrayString"`, `"ArrayVector"`, `"ArrayEntity"`

可兼容数字码：`1/2/4/8`。
